# phenop NEWS

## phenop 0.1.0 (2024-01-03)

### Initial Release

* First release of the phenop package
* Implements multidimensional plasticity analysis
* Includes specialized functions for host-pathogen interactions
* Provides visualization tools for reaction norms and plasticity landscapes
* Includes two example datasets: fungi_insect_data and plasticidad_ejemplo

### Features:
- `multidim_plasticity()`: Main function for multidimensional plasticity analysis
- `plot_reaction_norm()`: Visualization of reaction norms
- `host_pathogen_interaction()`: Analysis of host-pathogen interactions
- `anova_plasticity()`: ANOVA for plasticity effects
- `plasticity_meta_analysis()`: Meta-analysis tools

### Data:
- `fungi_insect_data`: Simulated insect-fungus interaction data
- `plasticidad_ejemplo`: Example dataset for plasticity analysis
